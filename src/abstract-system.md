# Abstract System

## [Ontology](./ontology.md)

A [system](./system.md) composed of [abstract](./abstract.md) [entities](./entity.md) rather than physical ones. An abstract system is a [conceptual system](./conceptual-system.md) designed to [model](./model.md), organize, or operate on [abstractions](./abstraction.md).

Where a [conceptual system](./conceptual-system.md) is any system made of [concepts](./concept.md), an abstract system emphasizes the deliberate construction of interrelated abstractions — a [language](./language.md), a framework, a formalism.

## [Axiology](./axiology.md)

Abstract systems enable [entities](./entity.md) to reason about [domains](./domain.md) without requiring physical instantiation. They are the substrate of [Domain Driven Design](./domain-driven-design.md): every [domain](./domain.md) model is an abstract system.

A programming language, a mathematical framework, a legal code, a [philosophy](./philosophy.md) — each is an abstract system that provides [coherence](./coherence.md) within its [scope](./scope.md).

### [Pattern](./pattern.md) Expression

Abstract systems exist anywhere [entities](./entity.md) organize [abstractions](./abstraction.md) into coherent wholes.

#### Mathematics

A mathematical framework (e.g. [category theory](./category-theory.md)) is an abstract system — axioms, definitions, and theorems forming a self-consistent whole with no physical referent.

#### Programming Languages

A programming language is an abstract system of syntax, semantics, and [type](./type.md) rules that enables [communication](./communication.md) between humans and machines.

#### Legal Systems

A body of law is an abstract system of rules, precedents, and principles that governs behavior within a [domain](./domain.md).
